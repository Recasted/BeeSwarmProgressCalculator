# BeeSwarmProgressCalculator
Track your Bee Swarm Simulator progress! Input your inventory, see progress toward items, and plan your grind. Features item search, progress display, PNG icons, and a credits tab with Discord &amp; GitHub links. Safe &amp; offline.

Usage

Open the Item Search tab and select the item you want to check.

Enter your current inventory in the Inventory Editor tab and press Save.

Check your progress on any item. The tool will show how much of each material you have and how much is remaining.

Disclaimer

This tool does not connect to Roblox servers and cannot read your live inventory. All data must be entered manually. It is safe to use and does not violate Roblox's terms of service.
